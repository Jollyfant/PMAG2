Palpole Implementation

This tool can be used to convert from geomagnetic directions at a particular site to virtual geomagnetic poles and vice versa. Input a site location in decimal notation and declination, inclination (longitude, latitude) pairs separated by a comma. Each pair of values must go on a new line.
