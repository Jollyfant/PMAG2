Paleomagnetism.org 2 - MagIC Interface

The following interface can be used to convert Paleomagnetism.org 2.0.0 data to the MagIC format. Multiple files can be selected and each will be associated with a unique location and all specimens are mapped as individual sites. Some additional contributor metadata is required.

MagIC Data Model Version: v3.0



Contribution Metadata

Metadata that describes the contributor of the MagIC file. You may include an optional DOI if this data is related to a publication.

Select Data

Select Paleomagnetism.org 2.0 collection files to be added to your MagIC file. If the file does not pass validation, return to the Interpretation Portal to fill in all required specimen attributes. Click Download MagIC file to download your database file. This file can be uploaded in the MagIC portal.
