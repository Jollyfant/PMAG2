Paleomagnetism.org 2 - Data Library

Specimen collections that were exported from Paleomagnetism.org 2 can be added to this library. Data sets will be published on this page and are openly available for anyone. All data collections, samples, specimens, and interpretations are assigned unique persistent identifiers (PID) that can be used to reference each digital object respectively. If you want to add your data to the collective library write us an e-mail on the contact page.

Forking copies an existing dataset from the library. This dataset can be modified and will hold a reference the original data collection.
