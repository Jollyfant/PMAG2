Paleomagnetism.org 2 - Data Library

Complete collections of specimens exported from the Paleomagnetism.org 2 interpretation portal can be added to this library. Multiple collections can be grouped in to a single coherent publication. All data sets published on this page are unrestricted and openly available for anyone through a public HTTPS API. All data publications, collections, specimens, and interpretations are assigned unique persistent identifiers (PID) that can be used to reference each digital object respectively.

The paleomagnetism.org data library is a community-sourced data repository that respects FAIR data management principles. If you would like to contribute your data set to the collective library submit your results on the submission page.
