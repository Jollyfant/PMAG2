Paleomagnetism.org 2 - Interpretation

Demagnetization data from many formats can be imported to Paleomagnetism.org. Select your input format and choose the files to read from your filesystem. Specimens can be appended to an existing collection by checking the respective Append checkbox. The available keyboard shortcuts can be discovered in the settings tab.

It is recommended to start your analyis on a logical collection of specimens that belong together (e.g. a site).





Interpretation - Great Circle Fitting

Interpreted great circle components can be fitted to directional set points by pressing 9. Components are fitted in the active coordinate system. The algorithm of McFadden and McElhinny, 1988 is used.
